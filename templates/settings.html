About Scheduler

The automatic scheduler periodically syncs data from various sources to the local database. Each sync type has its own configurable interval.

  • Fast playlist generation
  • Accurate tag filtering
  • Up-to-date channel lists
  • Current program guide data

Note: Only enabled accounts and sources are synced automatically.

Sync Types

IPTV Accounts:

  • Channel & category sync
  • Tag extraction (after sync)
  • FCC enrichment (after sync)

EPG Sources:

  • Provider XMLTV data
  • External XMLTV URLs
  • Schedules Direct

FCC Data:

  • US broadcast station info
  • Call signs & locations
  • Network affiliations
